```

Border:     #dbd1cc (light gray-brown)

At 50% opacity:
Result = (#f7f3f0 О 0.5) + (#dbd1cc О 0.5)
       ? #e9e2dd (soft blend)
```

**Result:** Border appears to blend INTO the background rather than sitting ON TOP of it.

## ??? Customization Guide

### Adjust Fade Distance

**Shorter fade (more abrupt):**
```css
transparent 0%, 
hsl(var(--border) / 0.8) 5%,  /* ? Start sooner */
hsl(var(--border)) 50%, 
hsl(var(--border) / 0.8) 95%, /* ? End later */
transparent 100%
```

**Longer fade (more gradual):**
```css
transparent 0%, 
hsl(var(--border) / 0.3) 20%, /* ? Start later */
hsl(var(--border) / 0.6) 50%, 
hsl(var(--border) / 0.3) 80%, /* ? End sooner */
transparent 100%
```

### Adjust Peak Opacity

**Subtler border:**
```css
hsl(var(--border) / 0.4) 50%  /* Lower peak opacity */
```

**Stronger border:**
```css
hsl(var(--border)) 50%  /* Full opacity at peak */
```
